Libraries
==================

FPy provides a standard library of numerical functions.

Overview
--------

The standard library is organized into the following modules:

- **base**: FPy without standard libraries
- **core**: Core numerical operations and utilities  
- **eft**: Error-free transformations for basic arithmetic operations
- **vector**: Vector operations including norms, dot products, and more
- **matrix**: Matrix operations including linear algebra routines
- **metrics**: Common error metrics and condition numbers
